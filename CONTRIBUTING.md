# Contributing to TourGuide

Thank you for your interest in contributing to TourGuide!

## How to Contribute

- **Open an Issue:** If you find a bug, have a question, or want to suggest an improvement, please open an issue on GitHub.
- **Pull Requests:**
  1. Fork the repository
  2. Create a new branch for your feature or fix
  3. Make your changes (keep it minimal and focused)
  4. Submit a pull request with a clear description
- **Templates & Docs:**
  - If you want to add or improve templates or documentation, please follow the existing style and structure.

## Code of Conduct

Please be respectful and constructive in all interactions.

## Project Philosophy

TourGuide is intentionally minimal and manual. Please avoid suggesting automation or code generation features. Contributions should focus on:
- Improving clarity
- Adding new templates/examples
- Enhancing documentation
- Fixing errors or typos
